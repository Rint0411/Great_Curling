Project Introduction

기본 사항
* 플랫폼 : 안드로이드
* 장르 : 스포츠, 캐주얼
특징
* 로컬 멀티플레이 방식
* 다양한 맵
* 스톤마다 존재하는 특별한 능력
* 맵마다 다른 특수 효과 / 방해 요소 ( 환경 현상 )
플레이 방식
1.플레이어 두명이 자신의 팀 색상을 정한다.
2.게임에서 선공 색상을 알려준다.
3.게임 내에서 능력을 가지고 있는 스톤을 선택하여 바꾸거나 기본 스톤을 선택한다.
4.스톤을 날리거나 설치하여 과녁 중앙에 최대한 스톤이 가도록 하거나
다음 턴의 상태가 중앙에 가지 못하도록 방해를 한다.
5.다음 색상의 팀으로 턴이 넘어가고 반복한다.
6.1라운드씩 팀당 3번씩 스톤을 조작하고 3라운드를 진행 후 게임이 끝이난다.//규칙 확인 후 수정
7.점수를 계산하고 승패를 확인한다.
